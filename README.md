# Segreteria Segidio

Applicazione web per gestione contatti, eventi, liste invitati e risposte.

Produzione: [https://archivio-segreteria.segidio.org](https://archivio-segreteria.segidio.org)

## Setup locale

```bash
npm run dev
```

Aprire [http://localhost:3000](http://localhost:3000).

## Contesto progetto

Leggere prima:

- `AGENTS.md`
- `PIANO_DI_LAVORO.md`, finche' presente

## Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Supabase

## Variabili

Copiare `.env.example` in `.env.local` e valorizzare le chiavi solo localmente.

Non committare file `.env`.

## Comandi

```bash
npm run lint
npm run build
```
